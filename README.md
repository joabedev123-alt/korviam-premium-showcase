# Korviam Alumínio

Especialistas em esquadrias de alumínio sob medida para projetos residenciais, comerciais e corporativos. Qualidade, precisão e alto padrão de acabamento em Campinas e região.

## Tecnologias

- React
- Vite
- Tailwind CSS
- Shadcn UI
- TypeScript

## Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```
